Questão 1

  1. O que é NVM?

    1. O que é NPM e para que o utilizamos?

      2. É uma ferramenta usada para gerenciar várias versões do Node.js em um único ambiente. É um script bash que permite aos usuários instalar, desinstalar e alternar entre diferentes versões do Node.js.

    2. O que o comando "npm install prompt-sync" realiza?

      3. É uma biblioteca que permite interagir com o usuário para obter entrada pelo console de forma síncrona.

    3. Em qual diretório devemos instalar uma biblioteca javascript?

      4. As bibliotecas JavaScript geralmente são instaladas no diretório do projeto em que está trabalhando

  3. Existe diferença entre o código que cria e o que faz uso de uma variável em python? Se sim explique essa diferença

    4. A principal diferença em relação ao JavaScript é que em Python, você não precisa declarar o tipo de variável

  4. Existe diferença entre o código que cria e o que faz uso de uma variável em javascript? Se sim explique essa diferença.

    5. AS variáveis [var] tem escopo de função e pode ser redeclarada, [let] tem escopo de bloco e pode ser reatribuída, e [const] tem escopo de bloco e não pode ser reatribuída.


    Questão 2

    Explique as diferenças e especificidades de variáveis criadas com os comandos: var/let/const

    const são constantes e não podem ser reatribuídas após a inicialização

    var têm escopo de função e podem ser redeclaradas dentro da mesma função.

    let têm escopo de bloco e não podem ser redeclaradas no mesmo escopo.


    Questão 3

    Diga o que será impresso na tela após executar cada um dos códigos abaixo:

    var x = 5; function fun() { var x = 6; console.log(x); } O resultádo será 5

    console.log(x); let x = 5; function fun() { let x = 6; console.log(x); }

    console.log(x); O resultádo será 5 pelo console log fora da função e 6 pelo console log dentro da função
